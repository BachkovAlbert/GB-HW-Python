Команда показывает усатановленные библиотеки в выбранном окружении:
pip freeze 

Команда для созания файла зависимиости:
pip freeze > requirements.txt

Команда для установки всех библиотек (которые использовались  в проекте) в выбранное виртуальное окружение:
pip install -r <название файла с библиотеками>
Пример(для данного проекта):pip install -r requirements.txt
